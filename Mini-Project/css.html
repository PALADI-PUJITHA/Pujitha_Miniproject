What is CSS?

CSS (Cascading Style Sheets) is used to control the style and layout of web pages. It separates content (HTML) from presentation (CSS).

Basic Syntax

A CSS rule consists of a selector and declarations:


selector {
    property: value;
}

Example:


h1 {
    color: blue;
    text-align: center;
}

Colors and Gradients

CSS provides various ways to define colors, including named colors, hex, RGB, HSL, and gradients:


body {
    background: linear-gradient(to right, red, yellow);
}

Positioning

CSS positioning allows you to control the placement of elements on the page:

  • Static (default): The element follows the document flow.
  • Relative: Positioned relative to its normal position.
  • Absolute: Positioned relative to its nearest positioned ancestor.
  • Fixed: Positioned relative to the viewport.

Grid Layout

CSS Grid Layout is a powerful tool for creating complex layouts:


.container {
    display: grid;
    grid-template-columns: 1fr 2fr;
    gap: 10px;
}

CSS Variables

CSS variables allow you to reuse values throughout your stylesheet:


:root {
    --primary-color: #4CAF50;
}
button {
    background-color: var(--primary-color);
}
