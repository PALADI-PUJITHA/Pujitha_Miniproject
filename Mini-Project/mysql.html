Introduction to MySQL

MySQL is an open-source relational database management system (RDBMS). It is widely used for managing structured data and supports a wide range of platforms. MySQL is known for its performance, reliability, and ease of use.

Installation

To install MySQL, download the installer from the official MySQL website and follow the installation steps. For Linux, you can use package managers like apt or yum.

Connecting to MySQL

You can connect to MySQL using the command-line client:


mysql -u username -p

Replace username with your MySQL username. Enter the password when prompted.

Creating a Database

Use the CREATE DATABASE statement to create a new database:


CREATE DATABASE my_database;

Replace my_database with your desired database name.

Creating Tables

Tables are the fundamental structures in MySQL. Use the CREATE TABLE statement to define a table:


CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Inserting Data

Use the INSERT INTO statement to add data to a table:


INSERT INTO users (name, email) 
VALUES ('John Doe', 'john@example.com');

Querying Data

Retrieve data from a table using the SELECT statement:


SELECT * FROM users;

You can use conditions to filter results:


SELECT * FROM users WHERE name = 'John Doe';

Updating Data

Use the UPDATE statement to modify existing data:


UPDATE users 
SET email = 'john.doe@example.com' 
WHERE name = 'John Doe';

Deleting Data

Remove data using the DELETE statement:


DELETE FROM users WHERE name = 'John Doe';

Joins

Joins are used to retrieve data from multiple tables:


SELECT orders.id, users.name 
FROM orders 
INNER JOIN users 
ON orders.user_id = users.id;

Indexes

Indexes improve query performance. Create an index using the CREATE INDEX statement:


CREATE INDEX idx_email ON users (email);
