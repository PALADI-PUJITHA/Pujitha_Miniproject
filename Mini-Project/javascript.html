JavaScript Course

Introduction

JavaScript is a versatile programming language used for web development to add interactivity to websites.

Basic Syntax

JavaScript syntax is the set of rules for how JavaScript programs are constructed.


console.log("Hello, World!"); // Print to the console

Comments

Comments in JavaScript:


// This is a single-line comment
/* This is a 
multi-line comment */

Variables

Variables are declared using var, let, or const:


let name = "John";
const age = 30;
var isStudent = true;

Operators

JavaScript supports various operators like arithmetic, comparison, and logical operators:


let sum = 10 + 20;
let isEqual = (10 === 20);
let isTrue = (10 > 5 && 20 > 10);

Data Types

Common data types in JavaScript:

  • String
  • Number
  • Boolean
  • Object
  • Array
  • Undefined
  • Null

Functions

Functions are blocks of reusable code:


function greet(name) {
    return `Hello, ${name}`;
}
console.log(greet("Alice"));

Strings

Strings are sequences of characters:


let greeting = "Hello, World!";
console.log(greeting.length);
console.log(greeting.toUpperCase());

Arrays

Arrays are used to store multiple values:


let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[1]);

Conditional Statements

Control the flow of your program:


if (age > 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

Looping Statements

Perform repetitive tasks:


for (let i = 0; i < 5; i++) {
    console.log(i);
}

JavaScript Objects

Objects store properties and methods:


let person = {
    name: "John",
    age: 30,
    greet: function() {
        console.log("Hello, " + this.name);
    }
};
person.greet();

Classes

Define reusable templates for objects:


class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    greet() {
        console.log(`Hello, ${this.name}`);
    }
}
let john = new Person("John", 30);
john.greet();

JavaScript HTML DOM

Interact with HTML elements:


document.getElementById("demo").innerText = "Hello, World!";

Browser BOM

Access browser objects:


console.log(window.innerWidth);
alert("Welcome to JavaScript!");

Web APIs

Utilize APIs like Fetch and Geolocation:


navigator.geolocation.getCurrentPosition(position => {
    console.log(position.coords.latitude);
});

AJAX

Communicate with servers:


fetch("https://api.example.com/data")
    .then(response => response.json())
    .then(data => console.log(data));

JSON

Exchange data between server and client:


let jsonData = '{"name": "John", "age": 30}';
let obj = JSON.parse(jsonData);
console.log(obj.name);

jQuery

jQuery simplifies JavaScript tasks:


$(document).ready(function() {
    $("#demo").text("Hello, jQuery!");
});

JavaScript Graphics

Work with graphics using Canvas:


let canvas = document.getElementById("myCanvas");
let ctx = canvas.getContext("2d");
ctx.fillStyle = "blue";
ctx.fillRect(0, 0, 150, 75);
