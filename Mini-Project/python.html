Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability. It supports multiple programming paradigms and has a wide range of applications.

Basic Syntax

A simple Python program:


print("Hello, World!")

Control Flow

Conditional Statements


x = 10
if x > 5:
    print("x is greater than 5")
else:
    print("x is 5 or less")

Loops


for i in range(5):
    print(i)

n = 0
while n < 5:
    print(n)
    n += 1

Matplotlib

Matplotlib is a library used for creating static, interactive, and animated visualizations in Python.


import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [10, 20, 25, 30, 35]

plt.plot(x, y)
plt.title('Line Graph')
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.show()

Machine Learning with Python

Python provides libraries like scikit-learn, TensorFlow, and PyTorch for machine learning.

Example: Linear Regression using scikit-learn


from sklearn.linear_model import LinearRegression
import numpy as np

# Data
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([1.5, 3.2, 4.8, 6.1, 7.3])

# Model
model = LinearRegression()
model.fit(X, y)

# Predictions
predictions = model.predict([[6]])
print("Prediction for 6: ", predictions)

Python and MySQL

Python can connect to MySQL databases using the mysql-connector-python library.


import mysql.connector

# Connect to MySQL
conn = mysql.connector.connect(
    host="localhost",
    user="yourusername",
    password="yourpassword",
    database="mydatabase"
)
cursor = conn.cursor()

# Query
cursor.execute("SELECT * FROM employees")
result = cursor.fetchall()
for row in result:
    print(row)

conn.close()

Python and MongoDB

Python can connect to MongoDB using the pymongo library.


import pymongo

# Connect to MongoDB
client = pymongo.MongoClient("mongodb://localhost:27017/")
db = client["mydatabase"]
collection = db["employees"]

# Insert Data
collection.insert_one({"name": "John", "age": 30})

# Query Data
for employee in collection.find():
    print(employee)
