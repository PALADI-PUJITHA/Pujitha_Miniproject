Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It provides the structure of a webpage, allowing you to create headings, paragraphs, links, images, and much more.

Basic HTML Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My First HTML Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

HTML Elements

HTML documents are made up of elements. An element is defined by a start tag, some content, and an end tag:

<tagname>Content</tagname>

HTML Attributes

Attributes provide additional information about an element. They are always included in the opening tag and usually come in name/value pairs:

<a href="https://example.com" target="_blank">Visit Example</a>

Links

Links are created using the <a> tag:

<a href="https://www.w3schools.com">Learn HTML</a>

Images

Images are added using the <img> tag. It requires the src attribute to specify the image source and the alt attribute for alternate text:

<img src="image.jpg" alt="Description of image">

Colors

Colors can be added to text and elements using the style attribute:

<p style="color:blue">This is blue text.</p>

Tables

HTML tables are used to display data in a tabular format:

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>30</td>
  </tr>
</table>

Lists

HTML supports ordered and unordered lists:

Unordered List:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Ordered List:

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

Inline and Block Elements

HTML elements can be classified as inline or block:

  • Inline elements: Do not start on a new line. Examples: <span>, <a>, <img>
  • Block elements: Start on a new line. Examples: <div>, <p>, <h1>
