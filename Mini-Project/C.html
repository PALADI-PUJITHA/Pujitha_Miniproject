C Programming Course

Introduction and History

C is a powerful general-purpose programming language developed by Dennis Ritchie in 1972 at Bell Labs. It has been the foundation for many modern languages and is widely used for system programming, embedded systems, and more.

Basic Syntax

The syntax of C is simple and structured. A basic C program looks like this:


#include 

int main() {
    printf("Hello, World!\n");
    return 0;
}

Structure of a C Program

A typical C program contains:

  • Preprocessor Directives
  • Global Declarations
  • Functions
  • Main Function

Comments

Comments are used to make the code more readable:


// This is a single-line comment
/* This is a 
multi-line comment */

Variables

Variables store data values:


int age = 25;
float height = 5.9;
char grade = 'A';

Data Types

C provides several data types:

  • int
  • float
  • char
  • double
  • void

Operators

C supports arithmetic, logical, and relational operators:


int sum = 10 + 20;
int isEqual = (10 == 20);
int result = (10 > 5 && 20 > 10);

Input and Output

Input and output in C:


#include 

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("You are %d years old.\n", age);
    return 0;
}

Conditional Statements

if-else


if (age > 18) {
    printf("Adult\n");
} else {
    printf("Minor\n");
}

Switch Statement


switch (grade) {
    case 'A':
        printf("Excellent\n");
        break;
    case 'B':
        printf("Good\n");
        break;
    default:
        printf("Try harder\n");
}

Looping Statements

For Loop


for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}

While Loop


int i = 0;
while (i < 5) {
    printf("%d\n", i);
    i++;
}

Functions

Functions are reusable blocks of code:


int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(10, 20);
    printf("Sum: %d\n", result);
    return 0;
}

Pointers

Pointers store memory addresses:


int x = 10;
int *ptr = &x;
printf("Value: %d\n", *ptr);

Structures

Structures are used to group variables:


struct Person {
    char name[50];
    int age;
};

Enums

Enums represent named constants:


enum Day {MON, TUE, WED, THU, FRI};
